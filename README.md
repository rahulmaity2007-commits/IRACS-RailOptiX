# IRACS-RailOptiX
**An Intelligent Decision Support System for Real-Time Railway Operations**

RailOptiX is a Python-based desktop application designed to assist railway controllers in optimizing train movements in real time. The system analyzes operational conditions and provides intelligent recommendations for train prioritization, crossings, overtakes, and loop-line allocations to improve punctuality, network throughput, and operational efficiency.

---

## 📌 Features

- 🚦 Real-time train movement monitoring
- 🛤️ Intelligent train prioritization
- 🔄 Crossing and overtaking recommendations
- 🚉 Loop-line allocation support
- 📊 Operational analytics and visualization
- 🔐 Secure login and user authentication
- 🖥️ Modern desktop GUI built with PySide6
- 🗄️ Database integration for train and infrastructure data
- 📈 Decision support for railway controllers

---

## 🛠️ Technology Stack

- **Language:** Python 3.x
- **GUI Framework:** PySide6 (Qt for Python)
- **Database:** SQLite (initially, scalable to PostgreSQL/MySQL)
- **Version Control:** Git
- **Repository Hosting:** GitHub

## 🚀 Getting Started
